## 设计风格
- **主色调**: 暖橙色、金黄色、深蓝色的黄昏配色
- **风格**: 简约、治愈、自然、城市景观
- **情感**: 放松、平静、治愈、转换心情

## 技术规格
- **原始尺寸**: 竖屏 (约 1080x1920)
- **建议桌面适配**: 居中裁剪 + 暗色遮罩
- **格式**: JPG (高质量压缩)
- **文件命名**: `bg_{序号}_{主题}_{描述}.jpg`

## 使用建议
1. 在休息模式弹窗中作为背景轮播
2. 添加半透明深色遮罩确保文字可读性
3. 切换动画：淡入淡出 1.5s
4. 轮播间隔：5-8 秒
